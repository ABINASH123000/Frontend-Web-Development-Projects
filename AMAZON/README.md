# Amazon Clone (HTML & CSS)

This project is part of my frontend web development learning journey. It is a static clone of the Amazon homepage created using only HTML and CSS. The goal of this project is to improve my understanding of web page structure, layout design, styling, and responsive design principles.

## Features

* Amazon-inspired user interface
* Navigation bar and search section
* Product categories and promotional sections
* Footer layout
* Responsive design using CSS

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was built for educational purposes only. It is not affiliated with or endorsed by Amazon. The objective is to practice frontend development concepts by recreating a well-known website design.

## Learning Outcomes

Through this project, I practiced:

* Semantic HTML structure
* CSS styling and layouts
* Flexbox and positioning
* Responsive web design
* UI replication and attention to detail
